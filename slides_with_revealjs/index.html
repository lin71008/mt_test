Quick slides with reveal.js

Author: Nick Lin

介紹

使用 reveal.js 來快速生成好看的簡報.

教學

1. 開一個新資料夾.

2. 下載 reveal.js 至資料夾中.

4. 添加簡報標題, 內容, ...


  • 如同 HTML 的寫法.
  • 提供兩層簡報結構, 以 <section> 來進行分段.
  • 提供段落功能, 以 <class="fragment來進行段落.
  • 提供程式碼風格區塊, 預設以 <pre><code class="html"> 來進行標示.
  • 支持 Markdown 風格, 使用 <section data-markdown> 後, 於 <script type="text/template"> 中進行編寫.

5. 挑一個喜歡的簡報風格, 轉場特效, ...


@ Reveal.initialize...
transition: 'slide'
// none/fade/slide/convex/concave/zoom

<link rel="stylesheet" href="black.css" id="theme">
// black.css, simple.css, sky.css, ...

6. 完成拉.

懶就直接都用 data-markdown
